Detailed Description

template<typename T>
class Array2D< T >

Stores data of any type T in a 2D array.

This class also provides a Fortran-like access operator () as well as a function to access the data in the array through a pointer that can be passed to F77 or C routines.

Data in the array with size = xsize_ * ysize_.

The data is stored with the fastest running index in the x-dimension and the slowest running index in the y-dimension. The indices in every dimension run from 0 to their respective "size-1"
